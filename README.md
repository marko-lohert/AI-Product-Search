# AI-Product-Search
A web app that uses AI to search for products in a database using plain English or an uploaded image.

# AI
- LLM running locally using Ollama (see appsettings.json for more information)
- Microsoft.Extensions.AI libraries are used for an easy communication between this C# app and LLM (running locally with Ollama)

# Database:
SQLite
(see appsettings.json for more information)

# Web
Blazor Web App (Interactive render mode:  Auto (Server and WebAssembly)
